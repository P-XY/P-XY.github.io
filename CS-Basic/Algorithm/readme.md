# 算法


算法复杂度（只看最高复杂度运算）：
- O(1) 常数复杂度
- O(log n) 对数复杂度
- O(n) 线性时间复杂度
- O(n^2) 平方
- O(n^3) 立方
- O(2^n) 指数
- O(n!) 阶乘

如何计算递归的时间复杂度（比如斐波那契的递归--展开--树--O(2^n) ）
