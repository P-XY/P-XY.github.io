2.Git配置

  1. Ubuntu下安装:

    $sudo apt-get install git

  2. 全局设置：

    $git config --global user.name "用户名“
    $git config --global user.email "用户email”

  3. 创建ssh秘钥：

    $ssh-keygen -t rsa -C "用户email"

  4. 在Github添加sshkey：

    $cd ~ #回到/home目录
    $ls -a# 会看到有一个.ssh文件夹，把里面的id_rsa.pub公钥添加到Github上
    $ssh-T git@github.com# 认证本地是否与Github完成连接

3.Git使用

  1. 两种方式使用：

    a. $git init 在本地创建Repository，然后$git push到Github。
    b. 在Github创建Repository，然后clone到本地。
    最好用和最常用的都是第二种方法，我们用的最多的估计也是clone这个命令

  2. Git支持多种传输协议:

    有ssh://,git://,http(s)://或只是一个用户名为前缀。

    我最常用的一种：

    $git clone git@github.com:Github用户名/Repository名.git

    上面这个命令只会克隆defult主分支。要clone指定分支：

    $git clone -b 分支名 git@github.com:Github用户名/Repository名.git

  3. 提交到Github：$git push

    要提交到指定分支：$git push origin 分支名
